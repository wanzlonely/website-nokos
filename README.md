# Walz Nexus - OTP Marketplace

Platform OTP multi-user dengan harga realtime dari RumahOTP.

Fitur:
- Harga & stok realtime + profit Rp1.000 otomatis
- Saldo per-user dengan Upstash Redis
- Login email OTP
- Webhook auto-credit deposit dan OTP

## Deploy ke Vercel

1. Upload folder ini ke GitHub
2. Import ke Vercel
3. Tambah Environment Variables dari .env.example
4. Deploy

## Webhook
Set URL webhook di RumahOTP: https://domainkamu.com/api/webhook
